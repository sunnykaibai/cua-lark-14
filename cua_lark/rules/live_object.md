# Live Object

Use:
- Distinguish live UI objects from content depicted inside screenshots, cards, thumbnails, link previews, or document previews.
- Click only real current controls, popups, drafts, list rows, and chat bubbles.
- Judge message tasks from real live chat objects, not conversation-list snippets or image previews.

Avoid:
- Finishing from matching text inside screenshots/previews/cards.
- Clicking controls depicted inside an attached screenshot or thumbnail.
- Treating an old Feishu screenshot attachment as the current app state.
