# Conversation Operations

Use:
- Operate on the requested conversation-list row for unread, mark/pin, and complete/done tasks.
- Keep row operations in the conversation list area, not the chat header menu.
- Use visible row controls, hover controls, context menus, or visible status icons.
- If only click/wait/scroll are allowed, use visible/hover row controls instead of right-click.
- If a current-run complete/done click makes the target row disappear or move, treat that as operation evidence unless an error is visible.
- If the target row is not visible before action, search or scroll the conversation list.

Avoid:
- Keep scrolling the conversation list after a successful complete/done action just because the completed row is no longer visible.
- Treat a conversation-list preview as evidence for chat-message completion.
- Send a chat message as a substitute for conversation unread/mark/complete operations.
