# Docs Body Edit

Use:
- Body editing targets the editor area below the document title and author/modified metadata, not the title field.
- Preserve existing relevant document content unless the instruction explicitly asks to replace all content.
- Add new test content at the document end, after the requested anchor, or in a clearly empty paragraph.
- If the exact requested text already exists, verify whether the current state satisfies the task before typing again.
- For local text or Markdown material, paste/type it only after visually opening the target document and focusing the correct body area.

Avoid:
- Type body content into the document title.
- Clear the whole document unless the task explicitly asks for replacement.
- Duplicate exact case-id text after it already appears in the correct body location.

Batch:
- Batch focus plus type only when the body insertion point is visually unambiguous and no dialog/menu is involved.
