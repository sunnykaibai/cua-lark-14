# Screenshot Overlay

Use:
- Screenshot border, toolbar, screenshot menu, scrolling screenshot, recording, OCR, or extract-text controls are overlay states.
- If screenshot capture is not the task, close the overlay before continuing; `hotkey(key='esc')` is usually safest.
- Continue only from the next visible live app state.

Avoid:
- Treating overlay controls as normal IM/Calendar/Docs/composer controls.
- Clicking screenshot/recording/OCR controls unless the user requested screenshot functionality.
- Composing or sending while an overlay blocks the live app.
